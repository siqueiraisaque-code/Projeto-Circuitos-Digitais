# Projeto-Circuitos-Digitais
Projeto de circuitos lógicos desenvolvido para a matéria de Circuitos Digitais no curso de Ciência da Computação na Universidade Federal do Cariri.
## Status Atual do Projeto

O circuito atual já conta com duas partes principais:
1. **Display de 7 Segmentos:** A estrutura base para a exibição visual dos dados no projeto.
2. **Comparador de Números:** Uma primeira versão/tentativa de lógica para comparar magnitudes numéricas, que ainda será refinada.
